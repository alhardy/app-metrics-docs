Enum SamplingType

Sampling avoids unbound memory usage, sampling is used to generate Histogram's from a reservoir of values

Namespace:App.Metrics
Assembly:App.Metrics.dll
Syntax
public enum SamplingType
Remarks

https://en.wikipedia.org/wiki/Reservoir_sampling

Fields

Name Description
Default

If set the default sampling type configured via the AppMetricsOptions will be used.
ExponentiallyDecaying

Sampling will be done with a Exponentially Decaying Reservoir.
HighDynamicRange

Sampling will be done with a A High Dynamic Range (HDR) Histogram. Note: The HDR Histogram implementation is in beta stage, some issues might still be present.
LongTerm

Sampling will done with a Uniform Reservoir.
SlidingWindow

Sampling will done with a Sliding Window Reservoir. A histogram with a sliding window reservoir produces quantiles which are representative of the past N measurements. More information about Exponentially Decaying Reservoir
