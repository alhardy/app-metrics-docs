Interface IMetrics

Provides access to record application metrics.

Namespace:App.Metrics
Assembly:App.Metrics.dll
Syntax
public interface IMetrics : IHideObjectMembers
Remarks

This is the entry point to the application's metrics registry

Advanced

Provides access to more advanced application metrics operations

Declaration
IAdvancedMetrics Advanced { get; }
Property Value
Type Description
IAdvancedMetrics
Remarks

- Provides access to the regsitered IClock used for timing - Allows retrieval of a snapshot of metrics data recorded via the IMetricsDataProvider and reseting the data at runtime - Allows retrieval of the current health of the application via the IHealthStatusProvider - Allows instantiation of a metric without performing the option - Allows disabling of metrics recording at runtime
