Class MetricItem

Metric items can be used with Counter or Meter MetricTypes

They provide the ability to track either a count or rate for each item in a counters or meters finite set respectively. They also track the overall percentage of each item in the set.

This is useful for example if we needed to track the total number of emails sent but also the count of each type of emails sent or The total rate of emails sent but also the rate at which type of email was sent.
